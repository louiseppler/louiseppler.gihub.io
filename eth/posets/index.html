Short Explanation

This partially ordered set (poset) is displayed as a Hasse Diagram.
In Addition a function f: A→A for the poset (A,≼) is defined. If a function from a node goes to itself, the arrow is not displayed. (To begin with f = id, thus no arrows are drawn.)

The function f is monotone if
∀ a,b ∈ A : a ≽ b => f(a) ≼ f(b)

A node a is a fixed point if f(a) = a and a post-fixed point if f(a) ≼ a.
The fix(f) is the set of all fixed points and red(f) is the set of all post-fixed points.
