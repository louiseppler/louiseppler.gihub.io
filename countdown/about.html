Countdown

Countdown is a British game show. This website implements the numbers puzzle from it. The goal is to compute a target number (100-999) with six numbers only using addition, subtraction, multiplication and division. From the six numbers, there are big ones (25,50,75,100 each once) and small ones (1-10, each twice) available.
This website allows you to generate those numbers and provides a solution to the puzzle.
