Sonic Boom

Initial Speed Acceleration

This animation shows what happens when an object (for example an airplane) travels close to the speed of sound. The circles show how the sound travels from the moving object. One can see the overlapping sound waves at the darker spots in the animation. This is when a sonic boom happens. For more detail see: Wikipedia
In this animation the speed of sound is 1. The acceleration is applied every frame (60 times per second)
