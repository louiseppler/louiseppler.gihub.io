Meeting Scheduler

Note: This page might be useful for other events. For example a sportsgame. Instead of people use teams and instead for rooms use field 1, 2 etc.

This website will help you plan meetings where every person has an individual meeting with every other person. In addition you can also limit the number meetings that happen in parallel. This option is useful if you have a limited number of rooms available.
The Algorithm is designed in a way that the necessary breaks are spread evenly per person. Limiting the number of meetings happening in parallel will result in more breaks.
